# 2.4_PA – Data Cleaning and Visualization Using Python

This project performs data cleaning, descriptive statistics, and data visualization on a portion of the Inside AirBnB 2022 listing dataset using Python in a Jupyter Notebook.

The program removes duplicates, fills missing values using the median, corrects capitalization inconsistencies, removes outliers, and saves a cleaned dataset. It also calculates descriptive statistics and generates visualizations including a histogram, bar chart, scatter plot, and heat map.

Tools used: Python, Pandas, NumPy, Matplotlib, and Jupyter Notebook.
